📄 How to Add Projects via Code (projects-data.js)

  1. Open the file projects-data.js in your editor
  2. Copy the template below and paste it into the PROJECTS_DATA array
  3. Fill in your project details and save the file
  4. The project will appear automatically on the portfolio
